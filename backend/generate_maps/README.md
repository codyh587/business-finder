#### Create a secrets.json file in this folder with the following info:

```json
{
    "BING_MAPS_KEY": "Bing Maps key",
    "DB_HOST": "DB host name",
    "DB_USER": "DB user name",
    "DB_PASSWD": "DB user password",
    "DB_NAME": "DB name",
    "DB_CERT": "DB SSL CA file name"
}
```
